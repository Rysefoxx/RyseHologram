# Dependency
_Make sure you shade the API._

### Gradle (Groovy) 
```
repositories {
    mavenCentral()
}
dependencies {
    implementation 'io.github.rysefoxx:Hologram:1.0'
}
```
### Gradle (Kotlin) 
```
repositories {
    mavenCentral()
}
dependencies {
    implementation("io.github.rysefoxx:Hologram:1.0")
}
```

### Maven
```xml
<dependency>
  <groupId>io.github.rysefoxx</groupId>
  <artifactId>Hologram</artifactId>
  <version>1.0</version>
</dependency>
```
